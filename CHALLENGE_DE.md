Programmier-Challenge: MVC-Muster und Admin-Panel hinzufügen

Implementieren Sie das MVC-Muster und erstellen Sie ein Admin Control Panel (ACP) zum Erstellen, Lesen, Aktualisieren und Löschen der Beiträge.

Anforderungen

- Verwenden Sie Nunjucks und Vanilla JS.
- Verwenden Sie das MVC-Muster.
- Verwenden Sie ein beliebiges CSS-Framework für den Admin-Bereich.
- Implementieren Sie eine Admin-Ansicht, die alle Blog-Einträge auflistet.
- Es sollte möglich sein, einen Eintrag zu bearbeiten und zu löschen.
- Es sollte möglich sein, neue Einträge hinzuzufügen.
- Daten sollten in der vorhandenen JSON-Datei gespeichert werden.
- Implementieren Sie einen WYSIWYG-Editor zur Bearbeitung des Hauptinhalts des Blog-Beitrags
  - Einfacher Editor <https://quilljs.com/>
  - Notion-ähnlich <https://editorjs.io/>
  - Oder schauen Sie hier <https://github.com/JefMari/awesome-wysiwyg-editors?tab=readme-ov-file#standalone>
- Speichern Sie HTML-Code immer bereinigt!

Optional

- Fügen Sie eine Suche für die Blog-Eintragliste hinzu
- Implementieren Sie eine grundlegende Authentifizierung für die Admin-Routen
- Fügen Sie Paginierung für Admin- und öffentliche Auflistung von Blog-Beiträgen hinzu
