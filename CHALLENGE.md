Coding Challenge: Add MVC Pattern and admin panel
Implement the MVC Pattern and create an Admin Control Panel (ACP) to create, read, update and delete the posts.

Requirements

- Use Nunjucks and Vanilla JS.
- Use the MVC Pattern.
- Use any CSS framework you want for the admin area.
- Implement admin view that lists all blog entries.
- It should be possible to edit and delete an entry.
- It should be possible to add new entries.
- Data should get stored into the existing JSON file.
- Implement a WYSIWYG Editor to edit the main content of the blog post
  - Simple editor <https://quilljs.com/>
  - Notion like <https://editorjs.io/>
  - Or take a look here <https://github.com/JefMari/awesome-wysiwyg-editors?tab=readme-ov-file#standalone>
- Store HTML code always sanitized!

Optional

- Add a search for the blog entry list
- Implement a basic authentication for the admin routes
- Add pagination for admin and public listing of blog posts
